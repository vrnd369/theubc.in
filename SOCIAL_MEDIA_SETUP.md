# Social Media Links Setup Guide

This guide explains how to add social media links to the website footer using the admin dashboard.

## Steps to Add Social Media Links

1. **Login to Admin Dashboard**
   - Navigate to your admin login page
   - Login with your admin credentials

2. **Go to Footer Management**
   - In the admin dashboard, navigate to **Footer Management**
   - The path is: `/admin/footer` (or `/superadmin/footer` for super admin)

3. **Add Social Media Links**
   - Scroll down to the **"Social Media Links"** section
   - Click the **"+ Add Social Link"** button

4. **Configure Each Social Media Platform**

   For each platform, you need to:
   
   a. **Select Platform**: Choose from the dropdown:
      - LinkedIn
      - Instagram
      - Facebook
      - Twitter
      - YouTube
      - Pinterest
      - TikTok
   
   b. **Enter Platform Name**: Enter the display name (e.g., "Facebook", "Instagram")
   
   c. **Enter URL**: Paste the full URL to your social media profile

5. **Add All Required Social Media Links**

   Use these URLs for your social media accounts:
   
   - **Facebook**: `https://www.facebook.com/soilkingfoods`
   - **Instagram**: `https://www.instagram.com/soilkingfoods/`
   - **YouTube**: `https://www.youtube.com/@soilkingfoods`
   - **LinkedIn**: `https://www.linkedin.com` (or your specific LinkedIn company page URL)

6. **Save Changes**
   - After adding all social media links, click the **"ðŸ’¾ Save Social Media"** button
   - Wait for the confirmation message

7. **Verify**
   - Visit your website footer
   - Check that all social media icons are displayed correctly
   - Click each icon to verify they link to the correct social media pages

## Notes

- The social media icons will automatically appear in the footer based on the platform you select
- You can add multiple social media links
- You can delete any social media link by clicking the "Delete" button next to it
- Changes are saved to Firebase and will appear on the live website immediately

## Troubleshooting

- If icons don't appear: Make sure you've selected the correct platform from the dropdown
- If links don't work: Verify the URL is complete and includes `https://`
- If changes don't save: Check your internet connection and try again
